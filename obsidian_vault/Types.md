Basic algebraic types:

1. Union type with `union` operator
2. Tuple types with `(type, ...)`
3. Record types with `(label: type, ...)`
4. Function types with `label: type -> type`
5. Nominal types with `nominal` operator
6. Unit type with `()`

Primitives:

1. `number`
2. `string`
3. `boolean`
4. `symbol`
